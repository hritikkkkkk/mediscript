## 📖 Overview

Medical reports often contain technical terminology that can be difficult for patients to understand. MediScript AI bridges this gap by leveraging Google's Gemini AI to explain medical reports in simple, patient-friendly language.

Users can upload or paste medical reports and instantly receive:

- Plain English explanations
- Disease and condition summaries
- Medical term definitions
- Suggested questions to ask a doctor
- Important health insights

> **Disclaimer:** MediScript AI is an educational tool and should not replace professional medical advice.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/mediscript.git

cd mediscript
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Development Server

```bash
npm run dev
```

## 🚀 How It Works

1. Upload or paste a medical report.
2. The report is securely processed.
3. Google Gemini analyzes the report.
4. AI generates an easy-to-understand explanation.
5. Users receive structured insights in seconds.
